# WJUIRoutable

http 请求组件api

### CocoaPods 安装

```
'pod WJUIRoutable'  or  'pod WJUIRoutable/API'  'pod WJUIRoutable/Core'
```

### 要求
* ARC支持
* iOS 6.0+


### 使用方法

* 在WJConfig配置中心：

```
//WJUIRoutable在配置中心名称
WJUIRoutable : {
    defaultNavigationController:(String)默认导航控制器名称
    routerURLformatter:(String)路由链接格式化器
}

```
